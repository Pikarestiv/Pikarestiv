## 🧑‍💻 About Me

I'm **Okwudili Joshua Odumodu** (Josh), a **Frontend/Mobile/Full-Stack Engineer** with **6+ years of experience**, based in **Nigeria 🇳🇬**. I'm the co-founder of **[Dumos Technologies](https://dumostech.com)**, a software consulting firm crafting scalable systems across fintech, agri-tech, edtech, and social.

- 🔭 &nbsp;Currently building **Bumpit** (dating app) and **Dumos LMS**, **DumosRx** and **Anmhris**
- 🌱 &nbsp;Leveling up in **AI/ML and System Design**
- 👨‍🏫 &nbsp;Mentored **15+ developers** - ask me anything about **Node.js, React Native or TypeScript**
- 🏗️ &nbsp;I architect systems with **clean architecture, microservices, and well-defined contracts**
- ⚡ &nbsp;Fun fact: I studied **Chemical Engineering** before jumping into code - spatial thinking is my secret weapon
- 🌐 &nbsp;Portfolio: **[okwycodes.dev](https://okwycodes.dev)** &nbsp;|&nbsp; Company: **[dumostech.com](https://dumostech.com)**
- 📫 &nbsp;Reach me: **okwyodumodu@gmail.com**
